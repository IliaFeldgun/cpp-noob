# cpp-noob

[![asciicast](https://asciinema.org/a/XcFyv1oBxaF7c32ePaNMg0XRx.svg?t=1)](https://asciinema.org/a/XcFyv1oBxaF7c32ePaNMg0XRx?t=0)

### Install using 
```bash
sudo apt-get update
sudo apt-get install build-essential manpages-dev
g++ --version
make --version
```
```bash
sudo dnf install gcc-c++
g++ --version
make --version
```

### Run using
```bash
make && ./main.out
```
