### Install using
```bash
sudo apt-get update
sudo apt-get install build-essential manpages-dev libncurses5-dev libncursesw5-dev
g++ --version
make --version
```
```bash
sudo dnf install gcc-c++
g++ --version
make --version
```

### Run using
```bash
make && ./main.out
```
