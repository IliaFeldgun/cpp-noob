# cpp-noob

[![asciicast](https://asciinema.org/a/d7DHHwwuyOVXfTimhSCzGlrRY.svg?t=1)](https://asciinema.org/a/d7DHHwwuyOVXfTimhSCzGlrRY?t=1)

### Install using 
```bash
sudo apt-get update
sudo apt-get install build-essential manpages-dev
g++ --version
make --version
```
```bash
sudo dnf install gcc-c++
g++ --version
make --version
```

### Run using
```bash
make && ./main.out
```
